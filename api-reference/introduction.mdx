## Welcome to the Voqo AI API

The Voqo AI API allows you to programmatically trigger outbound calls using your voice AI agents.

## Base URL

All API requests should be made to:

```
https://api.voqo.ai
```

## Authentication

All API endpoints require authentication using Bearer tokens. Include your API key in the `Authorization` header:

```bash
Authorization: Bearer YOUR_API_KEY
```

You can generate API keys from your workspace settings in the [Voqo AI Platform](https://platform.voqo.ai).
