On the dashboard, you can update your agent settings. You can change the agent's name, conversation template, post-call actions, and other settings.
## Agent Settings
In the agent settings section you can update
- Agent name
- Voice (male or female)
- Enable or disable the agent (if you disable the agent, it will not respond to calls)

## Conversation Settings
Under Conversation Settings, you can update the agent's behavior. 
- Conversation template: Choose from a list of conversation templates. 
- Preview the conversation template: You can previous the agent's opening sentence and how the agent will respond to the caller.

## Post-call SMS
Under post-call SMS, you can update the phone number that receives the summary message.

## Contacts

Enabling automatic contact creation will automatically create a contact for each caller. If the caller's name is identified during the conversation. 

Enabling automatic contact name updates will update the contact name if the caller's name is different from the name in the contact list.
